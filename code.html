Шаблоны и обобщенное программирование
Пример использования шаблонов для создания обобщенной функции.
import std.stdio; // Функция с шаблонным параметром auto sum(T)(T[] arr) { T result = 0; foreach (x; arr) { result += x; } return result; } void main() { int[] numbers = [1, 2, 3, 4, 5]; writeln("Сумма: ", sum(numbers)); }
Метапрограммирование: CTFE
Пример использования Compile-Time Function Execution (CTFE) для вычисления факториала во время компиляции.
import std.stdio; // Функция вычисления факториала auto factorial(int n) { if (n == 0) return 1; return n * factorial(n - 1); } void main() { // Вычисление во время компиляции enum fact5 = factorial(5); writeln("5! = ", fact5); }
